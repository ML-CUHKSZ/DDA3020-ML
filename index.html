DDA3020 Machine Learning

This course provides an introduction to the field of machine learning, covering fundamental concepts, algorithms, and applications. Students will learn various machine learning techniques, including supervised learning (Linear regression, Support Vector Machine), unsupervised learning (K-Means), and neural networks (MLP, CNN, RNN, Transformer). The course combines theoretical foundations with practical implementation, preparing students for advanced study and research in machine learning.
By the end of this course, students will be able to:
  • Understand the fundamental concepts and principles of machine learning.
  • Implement and apply various machine learning algorithms.
  • Analyze and solve real-world problems using machine learning techniques.
  • Evaluate the performance of machine learning models.

Course Information

Machine learning is a complex and evolving discipline. Our course will cover the fundamental concepts and algorithms in machine learning. Notably, Fundamental ≠ SIMPLE. Machine learning is built on several other foundation subjects, including probability, linear algebra, optimization, computer science, etc. In particular, the topics include:

  • Introduction of Machine Learning
  • Review of probability & Linear Algebra & Optimization
  • Linear regression
  • Logistic regression
  • Support Vector Machines
  • Decision Tree and Random Forest
  • Neural Networks I (MLP & CNN)
  • Neural Networks II (RNN & Transformer)
  • Over-fitting, Bias-Variance Trade-off
  • Performance
  • Introduction of Unsupervised Learning
  • K-Means
  • Mixture Models
  • Expectation Maximization (EM) algorithm
  • Principal Component Analysis (PCA)

Prerequisites

  • probability Theory
  • Linear Algebra
  • Optimization
  • Basic Usage of Python

Learning Outcomes

  • Knowledge: a) Students will understand fundamental concepts and know classic algorithms from machine learning; and b) Students will be able to determine which algorithms are suitable to solve certain problem and which are not.
  • Skills: a) Students could utilize these methods to solve some real-world classification and regression problems. and b) Students could effectively estimate the performance difference between different methods.
  • Valued/Attitude: a) Students will have a basic picture for machine learning, which can help them to learn advanced courses about machine larning or design novel methods.
